First Create some tutorials:
    POST /api/tutorials
        body:
            {
                "title": "",
                "description": ""
            }

Show all tutorials:
    GET /api/tutorials

– Create new Comments:
    POST /api/tutorials/[:id]/comments
        body:
            {
                "content": ""
            }

– Retrieve all Comments of specific Tutorial:
    GET /api/tutorials/[:id]/comments

– Delete all Comments of specific Tutorial:
    DELETE /api/tutorials/[:id]/comments

– Delete a Tutorial:
    DELETE /api/tutorials/[:id]
